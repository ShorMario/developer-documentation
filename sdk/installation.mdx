To install our SDK, you need to run one of the following commands:

<CodeGroup>

```bash npm
npm install @whop-apps/sdk
```

```bash yarn
yarn add @whop-apps/sdk
```

```bash pnpm
pnpm add @whop-apps/sdk
```

</CodeGroup>
