<Accordion title="Why aren't there any cookies set on developer mode">
  If you are using developer mode and cannot access the users token, this is
  likely because you are not using Whop Proxy. It is required when using
  developer mode as it allows you to make realtime updates and view them in your
  browser. For more information, click [**here**](/sdk/whop-proxy).
